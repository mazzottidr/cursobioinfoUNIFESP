Módulo III - Aplicação das ferramentas de bioinformática no estudo da expressão gênica

  • Exp1 – Análise da expressão gênica por meio da PCR quantitativa (PCR em tempo real)

[Tutorial](link para tutorial) / [Slides em PDF](link para slides)

  • Teoria: resumo da metodologia, métodos de análise de expressão gênica (ΔΔCt, curva padrão relativa), baseline, threshold, escolha de genes endógenos como normalizadores, análise estatística dos dados de expressão gênica (ΔCt? 2-ΔCt?, FC?)

  • Prática: Estudo de um problema – dois grupos: 1 gene alvo, 1 gene endógeno; Fornecer arquivos brutos e um script em R para formatar os dados brutos em um único arquivo pronto para a análise; calcular ΔCt, 2-ΔCt e FC (Excel) e realizar a análise estatística (SPSS)

  • Exp2 – Análise da expressão gênica por meio de microarrays

[Tutorial](link para tutorial) / [Slides em PDF](link para slides)

  • Teoria: resumo da metodologia, arrays de uma ou duas cores, exemplos de desenhos experimentais, controles de qualidade do array (distribuição das intensidades, matriz de correlação dos arrays), análise exploratória (análise dos componentes principais, cluster hierárquico não supervisionado), identificação dos genes diferencialmente expressos (diferentes métodos de análise) anotações gênicas e análises funcionais (pathways, networks, gene ontology, gene set enrichment analysis)

  • Prática: pipeline completo de uma análise de microarray de expressão entre dois grupos (controle de qualidade, análise exploratória, identificação dos genes diferencialmente expressos e análises funcionais), usando R e outros softwares de análise funcional (MetaCore)

  • Exp3 – Análise da expressão gênica por meio de NGS (RNAseq)

[Tutorial](link para tutorial) / [Slides em PDF](link para slides)

  • Teoria: como os dados brutos são gerados; discutir diferentes caminhos para fazer uma análise de expressão gênica por RNAseq; explicar contagem de reads

  • Prática: pipeline completo para análise de RNAseq fazendo alinhamento no transcriptoma de referência (controle de qualidade, alinhamento, contagem, variant calling e análise estatística) (fastqc, bwa, samtools, bedtools e R)

Módulo IV - Aplicação das ferramentas de bioinformática no estudo da metilação do DNA

  • Met1 – Análise da metilação do DNA

[Tutorial](link para tutorial) / [Slides em PDF](link para slides)

  • Teoria: resumo de metilação, conversão ou precipitação, métodos de análise de metilação (PCR-MSP, HRM, MALDI-TOF, Sanger, NGS, Pirossequenciamento) – focar em como é o resultado bruto para fazer as análise
