Estudos de Expressão Gênica por qPCR (PCR em tempo real)

Introdução

No Tutorial desta aula, vamos relembrar alguns conceitos aprendidos no Tutorial de Introdução ao R para criar um script capaz de fazer uma análise de expressão gênica de um gene alvo e usando um gene endógeno como normalizador, comparando-os entre dois grupos. A ideia neste tutorial é aplicar ferramentas do R para importar arquivos, fazer cálculos simples, usar testes estatísticos para comparar grupos, criar gráficos e exportar os resultados.

Esta aula será do tipo “Faça você mesmo“, onde este Tutorial conterá apenas dicas do que devemos fazer e que funções usar. Lembre-se de acompanhar o Tutorial do R para retomar as funções aprendidas.

Instruções “em Português” para o computador

Abaixo, vamos descrever em Português o que você precisará fazer. Seu objetivo é traduzir essas informações para a linguagem do R, de maneira a executar o script adequadamente.

Primeiramente, vamos fazer o download dos arquivos que contém os valores de Ct dos genes alvo e endógenos de cada grupo: Grupo A e GrupoB. Para que os arquivos não abram em uma página da internet, clique com o botão direito do mouse e escolha a opção Salvar link como…, e salve em uma pasta que depois você irá escolher como Working Directory do R.

Em seguida, monte o seu script de acordo com as seguintes instruções, nessa ordem:

  1. Definir o Working Directory
  2. Importar o arquivo grupoA.txt e grupoB.txt
  3. Calcular o Delta Ct (diferença entre gene alvo e gene endógeno) para cada grupo
  4. Calcular a média do Delta Ct de cada grupo e mostrar o resultado na tela (use a função print())
  5. Realizar a análise estatística usando a função t.test(x,y) onde xé o vetor com os valores de Delta Ct de um grupo e yé o vetor com os valores de Delta Ct do outro grupo, e mostrar o resultado na tela
  6. Calcular o Fold Change usando o grupo A como referência (usar a média do DeltaCt de cada grupo no cálculo) e mostrá-lo na tela
  7. Criar um boxplot com valores de 2^-DeltaCt, representando a expressão de cada grupo e salvar em um arquivo .pdf (Obs: use a ajuda do R ou o Google para entender como fazer um boxplot)

Dica 1: Use e abuse de comentários no seu script! Ele poderá ser usado para analisar seus próprios dados depois, desde que estejam formatados como os arquivos grupoA.txt grupoB.txt.
Dica 2: Não se esqueça de salvar cada resultado que criar em um novo objeto.

Ao terminar de montar o script, feche o RStudio e em seguida abra novamente e tente rodar seu script usando o comando source() (lembre-se de estar no mesmo diretório que o script).
