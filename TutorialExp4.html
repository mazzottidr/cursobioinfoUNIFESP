Objetivos do exercício

No exercício a seguir, iremos:

a) Construir uma rede de interação proteína-proteína a partir de um conjunto de dados obtidos do artigo de Zhao et al, 2013: “Association study of 167 candidate genes for schizophrenia selected by a multi-domain evidence-based prioritization algorithm and neurodevelopmental hypothesis”;
http://www.ncbi.nlm.nih.gov/pubmed/23922650

b) Identificar módulos e funções enriquecidas na rede de interação proteína-proteína;

c) Calcular medidas de centralidade nos nós presentes nessa rede;

d) Plotar os valores de medidas de centralidade para cada nó com o objetivo de se identificar hubs e bottlenecks na rede de interação.

Passo a passo

1) Utilizaremos para construção da rede o software Cytoscape, cujo download pode ser realizado a partir de seu website na internet. Após download e instalação do software, podemos acessar o programa a partir de seu ícone instalado no desktop.

2) Para instalar os plug-ins que utilizaremos, ir na aba Manage plug-ins e instalar os plug-ins GeneMania, MCode, Centiscape e NetworkAnalyzer. Para o GeneMania, é importante instalar as bases de dados referentes às espécies cujos dados de interação entre genes/proteínas desejamos acessar (“Install Data”).

3) Acessar o GeneMania no Cytoscape. Após acessá-lo, selecionar o organismo que se deseja obter as informações (“Choose an Organism”) imputar os dados dos genes ou proteínas que desejamos avaliar (“Choose Genes of Interest”). Selecionar também quais tipos de interação desejamos que estejam presentes na rede que será construída (“Choose Interaction Networks”. Rodar o GeneMania para construir a rede de interação.

4) Após remover as proteínas que não estão conectadas, determinar se a distribuição dos graus dos nós presentes na rede seguem uma lei de potência (scale-free network), utilizando o plug-in NetworkAnalyzer. Essa determinação pode ser realizada utilizando o comando “Fit Line”.

5) editar a rede de acordo com o que se deseja visualizar, utilizando o VizMapper. Pode-se alterar tamanho, formato e cor dos nós e arestas presentes na rede. O link a seguir possui informações detalhadas sobre as possibilidades de edição e visualização da rede utilizando o Vizmapper:
http://wiki.cytoscape.org/Cytoscape_User_Manual/Visual_Styles

6) Identificar módulos funcionais utilizando o plug-in MCode. Esse plug-in calcula os coeficientes de clusterização para cada nó e determinada a partir de um threshold determinado os módulos de nós presentes na rede.

7) Calcular as medidas de centralidade degree e betweenness utilizando o software Cytoscape.

8) Exportar resultados contidos no Data Panel a partir da aba File. Nessa aba, ir em Export Results para exportar os resultados da tabela.

9) Os dados podem ser posteriormente avaliados plotando os resultados de Node Degree e Node Betweenness para cada nó no excel, com o objetivo de se identificar Hubs e Bottlenecks entre os genes.

10) Funções hiper-representadas pelos genes identificados em cada módulo podem ser encontradas utilizando a ferramenta on-line Enrichr:
http://amp.pharm.mssm.edu/Enrichr/

O artigo referência dessa ferramenta se encontra a seguir:
Chen EY, Tan CM, Kou Y, Duan Q, Wang Z, Meirelles GV, Clark NR, Ma’ayan A. Enrichr: interactive and collaborative HTML5 gene list enrichment analysis tool. BMC Bioinformatics. 2013;128(14).
